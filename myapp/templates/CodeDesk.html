#include <stdio.h>
int soma_vetor(int v[], int n) {
    int total = 0;
    for (int i = 0; i < n; i++) {
        total += v[i];
    }
    return total;
}

Como organizar as pastas

O CodeGrader espera uma estrutura de pastas específica para associar cada submissão ao seu gabarito automaticamente.

1. Pasta raiz da prova
Crie uma pasta para a prova (ex.: prova-01/). Dentro dela ficam duas subpastas obrigatórias: submissoes/ e gabaritos/.
2. Submissões por aluno
Em submissoes/, crie uma pasta por aluno usando a matrícula ou o nome. Cada arquivo deve começar com um cabeçalho de identificação seguido do código.
3. Gabaritos por problema
Em gabaritos/, salve um arquivo por problema. O nome do arquivo deve corresponder ao identificador do problema presente no cabeçalho da submissão (ex.: p01).
